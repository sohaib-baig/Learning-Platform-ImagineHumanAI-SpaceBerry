Status: MVP – Free Beta (no payments)

User Stories

Auth: sign up using Google Sign-In, sign in using Google Sign-In, reset password

Welcome email after signup

Profile: view (name/email/country)

Dashboard: Today’s AI Read, Notice Board, counts (members/admins)

Classroom: list courses, filter by category

Course: overview, modules & lessons, begin/continue

Lesson: play/pause, speed, transcript, progress auto-save/resume, mark complete

Downloads: global & course-specific

Admin Stories

Admin auth (role-based access)

Courses CRUD (title, summary, category, hero)

Modules & lessons CRUD (order, transcript, resources, Mux playback id)

Publish/unpublish courses

Downloads CRUD (global/course)

Announcements: update Notice Board

Today’s AI Read: update

Admin dashboard stats: users count, admins count, courses count

Out of scope now: payments, enrolments, comments/community, Google login, certificates.
